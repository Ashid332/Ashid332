## 🚀 Featured Projects

### 🤖 DocuMind — LLM Document Q&A
A high-accuracy Retrieval-Augmented Generation (RAG) platform to query dense PDFs and technical documentation.
* **Stack**: LangChain · OpenAI · FAISS · FastAPI · Python
* **Impact**:
  * Automated document inspection, reducing query times from 15 minutes to 10 seconds.
  * Achieved 92% source citation accuracy with parent-document retrieval methods.
* **Link**: [GitHub Repository](https://github.com/Ashid332/documind)

### 📊 DataVault — Analytics Platform
A drag-and-drop web application that lets non-technical teams generate statistical charts from raw CSV files.
* **Stack**: Next.js · React · Node.js · Express · PostgreSQL · Tailwind CSS
* **Impact**:
  * Reduced internal analytics support tickets by 60% with self-service charting.
  * Designed non-blocking async stream parsers handling file sizes up to 100MB.
* **Link**: [GitHub Repository](https://github.com/Ashid332/datavault)

### 📉 Customer Churn Predictor
A predictive machine learning pipeline with local SHAP model explanations for customer retention.
* **Stack**: XGBoost · Scikit-learn · Pandas · FastAPI · Docker · Python
* **Impact**:
  * Trained classification models achieving 84% accuracy and an AUC-ROC of 0.81.
  * Integrated a local SHAP explainability layer serving predictions in under 50ms.
* **Link**: [GitHub Repository](https://github.com/Ashid332/customer-churn-predictor)
